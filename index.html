Project

Caballos En Libertad

Caballos en Libertad is a website created for a horse foundation in Buenos Aires, Argentina, aiming to highlight its mission and facilitate community interaction. The initiative came from the InkuA foundation, where I collaborate, and was a joint effort between the design area, which defined the aesthetics and usability, and the IT team, which carried out the technical implementation.

As a developer and leader of the IT team, I coordinated and contributed to the site's layout, ensuring the design translated into a fluid and accessible web experience. We used HTML, CSS, and JavaScript for content structuring and Bootstrap for responsive design, adapting to any device from mobile to desktops, maintaining a consistent and appealing aesthetic.

Project

Information Technology Information

Information Technology University is an administrative web application designed to simulate the management environment of a fictional university, facilitating the organization and administration of students, professors, courses, calendars, and classes. I fully conceived and developed it to apply and showcase my skills and knowledge in modern web development.

Using Angular 16 as the development core, I created a dynamic and highly interactive application that responds in real-time to user interactions. Angular Material UI made the interface intuitive and visually attractive, offering a smooth and coherent user experience. For data management, I chose Firestore, enabling effective and real-time data persistence, while Firebase Authentication handles user security and authentication.

Project

Café Cartagena

Café Cartagena is a specialized ecommerce offering an exclusive selection of coffees, marking my most ambitious project to date. Entirely conceived and developed by me, "Café Cartagena" simulates the operation of a virtual coffee company store, employing advanced web development techniques.

Built on Node.js and Express.js, and using MongoDB for data management, this project adopts a Server-Side Rendering (SSR) approach to optimize site delivery. The interface, created with Handlebars and styled with Bootstrap, ensures a consistent and adaptive user experience across all devices.

Project

World Forum

World Forum represents a personal challenge I undertook to deepen my experience in developing web applications using React.js and TypeScript. This project is not just a testament to my technical ability to create interactive web solutions but also reflects my commitment to adopting advanced web development practices, including the use of semantic HTML and meticulous project structure organization.

Entirely developed by me, "World Forum" serves as a proving ground to implement and refine my understanding of efficient folder and route organization in a React project, ensuring a scalable and maintainable codebase. This approach not only enhances code readability but also optimizes future collaboration and project adaptability.

Project

Nike Store

Nike Store is a SPA web application I designed and developed to simulate the user experience of an ecommerce site, focusing on the frontend presentation of an online store. This personal project was an opportunity to dive deeper into my skills with React.js, emphasizing custom component styling and REST API integration for dynamic data handling.

Choosing Firestore as the database management system aligns with the aim to implement realistic ecommerce functionalities, such as inventory management and real-time product updates. Through "Nike Store", I've showcased my ability to create intuitive and appealing user interfaces while efficiently managing business logic and data persistence in modern web applications.

Project

Human Front Aid

This website is currently being developed for the Human Front Aid foundation in Ukraine, backed by the InkuA foundation, where I am actively contributing as a developer. The project, which is still under construction, aims to provide multifaceted support amidst the armed conflict, utilizing HTML, CSS, JavaScript, and responsive design to ensure both accessibility and functionality.

My role in this collaborative effort has been pivotal, working alongside the design team to conceptualize and develop a platform that is not only intuitive and easy to navigate but also facilitates access to humanitarian assistance. Through the use of fundamental web technologies and responsive design practices, we are creating a digital solution that hopes to be a valuable resource for the coordination and distribution of aid in Ukraine.

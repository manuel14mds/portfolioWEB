Caballos En Libertad

Este sitio web fue creado para la fundación de caballos en Buenos Aires, Argentina, con el objetivo de destacar su misión y facilitar la interacción con la comunidad. La iniciativa provino de la fundación InkuA, en la que colaboro, y fue un esfuerzo conjunto entre el área de diseño, que definió la estética y usabilidad, y el equipo de IT, que llevó a cabo la implementación técnica.

Como desarrollador y líder del equipo de IT, coordiné y contribuí a la maquetación del sitio, asegurando que el diseño se tradujera en una experiencia web fluida y accesible. Utilizamos HTML, CSS y JavaScript para estructurar el contenido y Bootstrap para garantizar un diseño responsivo que se adapta a cualquier dispositivo, desde móviles hasta escritorios, manteniendo una estética coherente y atractiva.

Este proyecto no solo fortaleció mis habilidades en desarrollo web y liderazgo de equipo, sino que también me permitió contribuir a una causa significativa, apoyando la visibilidad y el impacto de una fundación dedicada a la protección y bienestar de los caballos.

Red nacional de Cuidados

Esta sitio permite que las personas que entregan asistencia permanente a quienes tienen una discapacidad, dependencia funcional moderada o severa y/o necesidades educativas especiales permanentes de todo chile, se registren en esta plataforma y tengan acceso a recursos de apoyo, videos en la sección de academia y participen en el Foro donde tienen información de todo chile o filtrada por localización.

Es una plataforma desarrollada con Next.js y Express.js como principales tecnologías, autogestionable con tres niveles de acceso(cuidador, administrador y root).

Information Technology University

Este proyecto consiste en una SPA (Single Page Application) administrativa diseñada para simular el entorno de gestión de una universidad ficticia, facilitando la organización y administración de estudiantes, profesores, cursos, calendarios y clases. Fue concebido y desarrollado íntegramente por mí como un medio para poner en práctica y demostrar mis habilidades y conocimientos en desarrollo web moderno.

Utilizando Angular 16 como eje central del desarrollo, logré crear una aplicación dinámica y altamente interactiva que responde en tiempo real a las interacciones del usuario. Gracias a Angular Material UI, la interfaz es intuitiva y visualmente atractiva, ofreciendo una experiencia de usuario fluida y coherente. Para la gestión de datos, opté por Firestore, lo que permite una persistencia de datos eficaz y en tiempo real, mientras que Firebase Authentication se encarga de la seguridad y autenticación de usuarios, garantizando un acceso seguro y personalizado a la plataforma.

Este proyecto no solo es un testimonio de mi capacidad para desarrollar soluciones de software completas e integradas, sino que también refleja mi compromiso con la creación de aplicaciones web modernas y seguras. Representa un ejemplo práctico de cómo las tecnologías de vanguardia pueden utilizarse para abordar y simplificar complejidades administrativas y educativas en un entorno virtual.

Café Cartagena

Este ecommerce especializado ofrece una selección exclusiva de cafés, representando mi proyecto más ambicioso hasta la fecha. Concebido y desarrollado íntegramente por mí, "Café Cartagena" simula la operación de una tienda virtual para una empresa ficticia de café, poniendo en práctica avanzadas técnicas de desarrollo web.

Construido sobre Node.js y Express.js, y utilizando MongoDB para la gestión de datos, este proyecto adopta el enfoque de Server-Side Rendering (SSR). La interfaz, creada con Handlebars y estilizada con Bootstrap, asegura una experiencia de usuario coherente y adaptativa en todos los dispositivos.

Este proyecto personal no solo me permitió aplicar un amplio conjunto de conocimientos y habilidades en tecnologías web, sino que también representó un desafío significativo, consolidando mi experiencia en el desarrollo de soluciones ecommerce completas y efectivas.

Nike Store

Es una aplicación web SPA que diseñé y desarrollé con el propósito de simular la experiencia de usuario de un ecommerce, específicamente enfocado en la presentación frontal de una tienda en línea. Este proyecto personal fue una oportunidad para profundizar en mis habilidades con React.js, poniendo especial énfasis en el estilado personalizado de componentes y la integración con APIs REST para manejar datos dinámicamente.

La elección de Firestore como sistema de gestión de base de datos se alinea con el objetivo de implementar funcionalidades realistas de ecommerce, como la gestión de inventario y la actualización de productos en tiempo real. A través de "Nike Store", he demostrado mi capacidad para crear interfaces de usuario intuitivas y atractivas, al tiempo que manejo eficientemente la lógica de negocio y la persistencia de datos en aplicaciones web modernas.
